# Generic AI Coding Agent Pack

This project contains reusable instructions for AI coding assistants.

## Phase model

1. Spec
2. Plan
3. Build
4. Test
5. Review
6. Ship

## Agent expectations

- Think like a senior engineer.
- Keep work structured.
- Avoid unbounded exploration.
- Prioritize correctness and maintainability.
- Make assumptions visible.
- Ask only necessary questions.
- Prefer practical delivery over theoretical perfection.
