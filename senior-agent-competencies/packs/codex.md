# Codex Agent Pack

Use this repository as the operating system for engineering tasks.

## Rules

- `/spec` defines requirements before building.
- `/plan` decomposes work before editing files.
- `/build` implements the smallest useful version.
- `/test` validates important behavior.
- `/review` finds bugs, security risks, and missing tests.
- `/ship` prepares docs, release notes, and final checks.

## Engineering standard

Apply these competencies by default:

- Clean Code
- Test-Driven Development
- Security by Design
- Error Handling
- Code Review
- Agent Safety
- Maintainability

## Behavior

Act like a senior engineer:
- clarify only when needed
- use minimal context
- avoid destructive changes
- protect secrets
- prefer simple architecture
- document tradeoffs
- produce reviewable changes
