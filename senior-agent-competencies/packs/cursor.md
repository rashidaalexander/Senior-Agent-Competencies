# Cursor Rules: Senior Agent Competencies

When the user types a slash command, follow the matching file in `commands/`.

## Slash commands

- `/spec`: use `commands/spec.md`
- `/plan`: use `commands/plan.md`
- `/build`: use `commands/build.md`
- `/test`: use `commands/test.md`
- `/review`: use `commands/review.md`
- `/ship`: use `commands/ship.md`
- `/secure`: use `commands/secure.md`
- `/ui`: use `commands/ui.md`
- `/api`: use `commands/api.md`
- `/debug`: use `commands/debug.md`
- `/refactor`: use `commands/refactor.md`
- `/docs`: use `commands/docs.md`

## Default standards

Always consider:
- security
- accessibility
- tests
- maintainability
- performance
- documentation
- human reviewability
